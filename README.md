# uniapp 小程序脚手架

## vite,vue3,ts

| 类别        | 库                                        |
| ----------- | ----------------------------------------- |
| 包管理器    | pnpm                                      |
| 网络请求    | uni-ajax                                  |
| 原子样式    | unocss                                    |
| 代码校验    | husky, lint-staged, eslint, prettier      |
| 状态管理    | pinia, pinia-plugin-persist-uni           |
| 类型导入    | types-sync, unplugin-auto-import          |
| 时间处理    | dayjs                                     |
| @uni-helper | eslint, uni(esm), manifest, pages, unocss |

## 使用

```
npx degit ijntvwh/uniapp_vue3_ts my-project

```
